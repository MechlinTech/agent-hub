# Dev Scaffold Agent

The Dev Scaffold Agent scaffolds projects on **your local machine** via the Local Executor.

## Prerequisites

- AgentHub Desktop installed and running
- Browser on the **same machine** as Desktop
- Account paired (Local Executor settings / connect flow)

## Quick start

1. Install and open AgentHub Desktop
2. Pair with your Agent Hub account
3. Confirm Local Executor shows connected
4. Open **Agents → Dev Scaffold → New Setup**
5. Configure stack → Review → Generate

## Notes

API routes and internal IDs remain under `project-setup` for compatibility.
